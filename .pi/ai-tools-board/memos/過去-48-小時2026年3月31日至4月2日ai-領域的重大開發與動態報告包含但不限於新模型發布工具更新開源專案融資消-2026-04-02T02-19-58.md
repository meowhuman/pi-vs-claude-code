|------|---------|---------|

**MCP 協議普及度**：Claude Code 的 hook 系統（defer hooks、PermissionDenied hook）和 Codex 的 spawn v2 inter-agent API 都在強化 agent 間的**程式化協調能力**，這比 MCP 更底層。Gemini CLI 則直接採用 A2A 協議。**開源 agent 生態確實被洩漏重塑了** — 但重塑方向不是「複製 Claude Code」，而是「四條路線各走各的」。

## 🔴 信號二：算力稀缺時代來臨，直接衝擊 Agent 成本模型

OpenAI $122B 融資 + 關閉 Sora + Oracle 裁 30,000 人 — 三件事同週發生，傳遞同一信號：**GPU 是新的石油，所有人都在搶**。

對 coding agent 的具體影響：
- **短期**：API 限速可能更嚴格（OpenAI 已展示願意犧牲百萬用戶產品換算力）
