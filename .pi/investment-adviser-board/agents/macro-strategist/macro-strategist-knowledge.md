## 核心投資哲學

1. **宏觀是底層框架** — 技術面和基本面都必須在正確的宏觀背景下才有意義。不要在通膨加速的環境中做多長期債券，也不要在 Fed 寬鬆週期中做空成長股。
2. **多維度驗證** — 單一指標不構成交易依據。需要跨資產信號（殖利率 + 美元 + 原油 + 黃金）交叉驗證。
3. **區分「噪音」與「訊號」** — X 上 90% 是情緒和 self-promotion。只追蹤有數據支撐、有可測試論點的分析師。
4. **週期定位優先於方向預測** — 先判斷我們處於經濟週期的哪個階段（擴張/頂部/收縮/底部），再決定風險偏好。
5. **地緣政治是黑天鵝的飼養場** — 2025-2026 的核心宏觀風險是伊朗衝突 → 原油供應中斷 → 通膨二次攀升。

### 跨資產信號儀表板

| 信號 | 多頭意義 | 空頭意義 |
|------|----------|----------|
| 10Y UST 殖利率 ↑ | 經濟增長預期升溫 | 通膨預期 / 財政風險溢價 |
| DXY ↑ | 美國資產吸引力強 | EM 資本外流、債務壓力 |
| Brent 原油 ↑ | 需求復甦 | 供給衝擊 / 地緣風險 |
| 黃金 ↑ | 避險需求 | 法幣信心下降 |
| VIX ↑ | 恐慌 → 可能是買入機會 | 真實風險事件 |

### 宏觀情境分析路徑

1. **央行路徑** — Fed / ECB / PBoC 下一步？市場定價 vs. 我的預期差異？
2. **數據路徑** — CPI / PCE / NFP / PMI 的 trend（不只是單月數字）
3. **地緣路徑** — 哪些地緣事件有資產定價影響？（當前：伊朗-美國、中美貿易）
4. **資金流路徑** — 資金在流進還是流出？EM 儲備變化？ETF 資金流？

### 商品監控分層邏輯

> 每個加入的 ticker 必須回答一個獨立的宏觀問題。

| Ticker | 商品 | 宏觀問題 | 與其他商品的獨立性 |
|--------|------|---------|------------------|
| **USO** | 原油 | 地緣供給衝擊有多嚴重？ | 供給驅動，與需求端完全不同邏輯 |
| **GLD** | 黃金 | 市場恐慌程度 / 法幣信心？ | 與油負相關（避險 vs. 風險），獨立信號 |
| **CPER** | 銅 | 全球實體經濟在增長還是衰退？ | 需求驅動，與油（供給驅動）形成對比 |
| **UNG** | 天然氣 | 歐洲能源安全 + 美國國內通膨？ | 與原油有區域差異 |
| **DBA** | 農業綜合 | 全球食品通膨趨勢？ | EM 央行政策空間，變化慢但影響大 |
| **WEAT** | 小麥 | 糧食安全？ | 俄烏/中東衝突時變重要 |

**關鍵組合信號：**
- **USO ↑ + CPER ↓** = 供給衝擊型滯脹（最差情境）→ 2026-04 正是此狀態
- **USO ↑ + CPER ↑** = 需求驅動型復甦（好情境）
- **GLD ↑ + USO ↑** = 真正的恐慌 / 法幣信心危機
- **UNG ↑ + USO ↔** = 歐洲能源危機獨立升級（俄羅斯管道問題）
- **DBA ↑ + GLD ↓** = 食品通膨但避險資金不進金 → EM 壓力 + 流動性緊張

### Gromen 情緒週期模式 [src:013]

2026-03-28 觀察到：連續 3 週出現「週一好消息（TACO）→ 週三糟糕 → 週五垃圾火」模式。

**可能用法**：
- 如果週一出現正面新聞 → 考慮對沖或短線做空
- 這不是鐵律，但作為短線情緒指標值得追蹤
- 需要更多週數驗證是否為持續模式

### Pettis 雙邊限制無效論 [src:014]

- 美中商品貿易逆差縮小 32% 至 $2,020 億
- 但整體商品貿易逆差創紀錄 **$1.24 兆**
- 進口商只是轉移到其他市場（越南、墨西哥等）
- **結論**：雙邊貿易限制是完全無效的政策工具
- Joan Robinson 預言：beggar-thy-neighbor 的報復終會來，但報復方式不一定理性

## 經驗教訓 / 避免的偏誤

1. **宏觀敘事偏誤**：Luke Gromen 式的大敘事非常有吸引力，但容易讓人忽略時間點。大方向對不代表交易時機對。
2. **確認偏誤**：追蹤的分析師可能都有自己的偏見。Bianco 偏空、Schiff 極空。必須主動尋找反向論點。
3. **數據 vs. 敘事**：在情緒高漲時（如地緣危機），市場由敘事驅動。在平靜時期，數據更重要。要識別當前市場由什麼驅動。
4. **供應鏈思維**：Pettis 和 Gromen 都在強調供應鏈的脆弱性。不能只看最終需求，要追蹤中間品和原材料流動。
5. **「自我預測」陷阱**：MacroAlf 提醒自己的 win rate 只有 ~50%。再好的宏觀分析師也不是算命師。核心能力是框架，不是預測。
6. **工具盲區陷阱** [src:015]：FRED 是美國經濟數據庫，不是全球央行數據庫。驗證 KOL 論點時，先查表確認工具能不能做到，不要假設能拿到數據。sta-v2 的外匯功能填補了這個缺口。
7. **EM 壓力 ≠ DXY** [src:016]：DXY 下跌不代表 EM 貨幣安全。2026-03 實證：DXY -4.4% 但 USDTRY +17.3%、USDINR +10.2%。石油進口國的壓力來自經常帳惡化（結構性），不是美元走勢（週期性）。
8. **黃金暴跌 ≠ 安全** [src:017]：在戰爭 + 滯脹環境中，GLD 暴跌 >20% 是異常信號。可能原因：被迫賣出（margin call）、去槓桿、流動性緊張。不能假設「避險資產一定漲」。需要追蹤 DBC 成交量（3.48x）和 EM 資金流來驗證。
9. **食品通膨是滯脹的第二波** [src:017]：DBA RSI 73.7 超買 + WEAT 偏多，在原油供給衝擊後，農業商品可能是下一個通膨壓力點。對 EM 央行和社會穩定影響更大。
