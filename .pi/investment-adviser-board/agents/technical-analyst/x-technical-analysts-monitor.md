## 注意事項

1. **不盲從單一分析師**: 每位分析師都有失誤，交叉驗證是關鍵
2. **注意時間框架**: 短線分析師 (Raschke, Redler) 與長線分析師 (Brandt, Kimble) 信號可能衝突
3. **記錄勝率**: 追蹤分析師近期呼叫的準確度，調整權重
4. **避免 confirmation bias**: 不要只讀與自己觀點一致的分析師
