## 建議的委員監控分工

### Position Manager
- 作為主責人
- 任務：維護整體倉位地圖、跨券商彙總、集中度、加減倉優先順序
- 特別應監控：VOO / XLK / TSLA / 中概（09988）/ 商品（GLD, SLV）/ 現金比例

### Risk Officer
- 任務：監控總風險、單一標的集中度、跨資產相關性、最大回撤、槓桿／期貨曝險
- 特別應監控：
  - TSLA + XLK + VOO + SPY 的美股 beta 疊加
  - GLD / SLV 商品波動
  - DX 期貨空單風險
  - 阿里巴巴單一港股集中度

### Technical Analyst
- 任務：監控所有可交易持倉的技術位與 action trigger
- 優先順序：TSLA、IONQ、OKLO、XLK、XLY、SLV、GLD、SPY、VOO

### Macro Strategist
- 任務：監控宏觀驅動與跨資產敘事
- 對應資產：
  - DX：美元方向
  - GLD / SLV：實質利率、避險需求
  - INDA：印度資金流與成長敘事
  - 中概 / 阿里巴巴：中國政策與監管週期
  - 美股大盤 ETF：Fed / 就業 / 成長與通膨路徑

### Market Intelligence
- 任務：追蹤即時新聞、催化劑、情緒變化
- 優先標的：TSLA、IONQ、OKLO、GOOG、AMZN、阿里巴巴

### Fundamental Analyst
- 任務：追蹤個股基本面與估值變化
- 優先標的：AMZN、GOOG、TSLA、IONQ、OKLO、阿里巴巴、友邦保險

### Backtest
- 任務：回測你目前常見交易風格是否有效
- 建議策略池：
  - TSLA / IONQ / OKLO 的 swing momentum / pullback
  - XLK / VOO / SPY 趨勢追蹤與再進場規則
  - GLD / SLV 反轉與突破策略

### CEO
- 任務：每次會議把以上監控資訊整合成兩件事：
  1. 現在要做什麼
  2. 若市場變化，要怎麼反應
